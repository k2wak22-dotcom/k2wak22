# 🏫 학급 자리배치 프로그램 (Class Seating Randomizer)

## 📖 프로젝트 개요
교사들이 학급 학생들의 자리를 쉽고 재미있게 배치할 수 있도록 돕는 웹 애플리케이션입니다. 학생 명단을 등록하고, 단순한 랜덤 배치를 넘어 '룰렛', '카드 뽑기' 등 창의적이고 흥미로운 애니메이션 요소를 결합하여 조례/종례 시간의 소소한 이벤트로 활용할 수 있습니다.

---

## ✨ 주요 기능 (Core Features)

### 1. 학생 명렬 관리 (Student List Management)
* **명단 입력:** 텍스트 영역(Textarea)을 통해 엑셀 등에서 복사한 이름/번호를 한 번에 붙여넣기 (콤마 또는 줄바꿈으로 구분).
* **수정 및 삭제:** 등록된 학생의 이름을 개별적으로 수정하거나 삭제.
* **데이터 저장:** 브라우저의 `LocalStorage`를 활용하여 새로고침해도 명단이 유지되도록 구현.
* **결석생 처리:** 당일 결석한 학생을 선택하여 자리 뽑기에서 제외하는 기능.

### 2. 교실 자리 배치도 설정 (Seat Layout Configuration)
* **가로/세로 줄 수 설정:** 학급 환경에 맞춰 자리 배치도 형태 지정 (예: 6x5, 4x8 등).
* **빈 자리 지정:** 기둥, 교탁 위치 등에 의해 실제 존재하지 않는 자리를 클릭하여 '비활성화(X)' 처리.
* **고정석 지정:** 시력 문제 등으로 앞자리에 꼭 앉아야 하는 학생을 특정 자리에 미리 고정.

### 3. 창의적인 자리 뽑기 모드 (Creative Draw Modes)

* 🎲 **랜덤 셔플 (초스피드 모드):**
    * 버튼 클릭 한 번에 전체 학생의 자리가 0.5초 만에 무작위로 배치됩니다.
    * *효과:* 화면이 흔들리는 애니메이션 후 결과 출력.
* 🎡 **룰렛 뽑기 (긴장감 모드):**
    * 학생 이름이 적힌 룰렛이 돌아가고, 당첨된 학생이 남은 자리 중 하나에 무작위로(또는 교사가 클릭한 자리에) 배치됩니다.
    * *효과:* 휠 회전 애니메이션 및 효과음.
* 🃏 **카드 뒤집기 (선택 모드):**
    * 화면에 자리 개수만큼의 뒷면 카드가 깔립니다. 학생이 앞으로 나와 원하는 카드를 클릭하면 본인의 자리가 공개됩니다.
    * *효과:* CSS 3D Transform을 이용한 카드 뒤집기 애니메이션.

## 🛠️ 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
    * *Framework (선택):* React.js 또는 Vue.js (상태 관리가 많아 프레임워크 사용을 권장)
    * *Styling:* Tailwind CSS 또는 Styled-components
* **Animation:** CSS Animations, `canvas` (룰렛 구현 시)
* **Storage:** Web LocalStorage (서버 없이 프론트엔드 단독 실행 가능하도록 설계)

---

## 🖥️ 화면 구성 (UI/UX Flow)

1.  **[명렬표 탭]:** 학생 이름 텍스트 박스 입력 -> "등록" 버튼 -> 리스트 UI로 확인
2.  **[자리 세팅 탭]:** 가로/세로 N칸 입력 -> 빈자리/고정석 마우스로 클릭하여 설정
3.  **[뽑기 모드 탭]:** * 상단: '초스피드 랜덤', '룰렛 뽑기', '카드 뒤집기' 탭 전환 버튼
    * 중앙: 시각적인 칠판 배경의 교실 UI
    * 하단: "뽑기 시작!" 버튼 및 초기화 버튼

## 🚀 구현 단계 (Roadmap)

* **Step 1: UI 마크업 및 기본 로직**
    * HTML/CSS로 칠판, 책상 레이아웃 잡기
    * JavaScript로 배열(Array)을 이용한 명단 저장 및 셔플(Fisher-Yates 알고리즘) 함수 구현
* **Step 2: 로컬 스토리지 연동**
    * 입력된 학생 명단을 브라우저에 저장하여 재접속 시 불러오기
* **Step 3: 뽑기 모드 상세 구현**
    * 룰렛: HTML5 Canvas의 `arc` 메서드와 `requestAnimationFrame`을 이용해 휠 애니메이션 구현
    * 카드: CSS `transform: rotateY(180deg);` 속성 적용
* **Step 4: 디테일 및 효과음 추가**
    * 자리 배치가 완료되었을 때 축하 꽃가루(Confetti) 효과 추가
    * 버튼 클릭 및 룰렛 회전 시 효과음(Web Audio API) 삽입
* **Step 5: 배포**
    * GitHub Pages 또는 Vercel을 이용한 무료 정적 호스팅 배포

## 💡 추가 아이디어 (Optional Features)
* **짝꿍(남녀) 섞기:** 남학생 명단과 여학생 명단을 분리하여, 강제로 남녀 짝꿍이 되도록 배치하는 옵션.
* **테마 변경:** 교실 모드, 우주 모드, 숲속 모드 등 배경과 책상 이미지를 변경할 수 있는 재미 요소 추가.
* **자리표 출력:** 완성된 자리 배치도를 PDF로 저장하거나 프린터로 바로 출력(Print API)하는 기능.
